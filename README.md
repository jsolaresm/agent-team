# Agent Team

Equipo de agentes IA orquestados con OpenClaw.

## Agentes
- **Orchestrator** — Analiza y delega tareas
- **Backend Dev** — APIs, bases de datos, arquitectura
- **Frontend Dev** — UI, React/Vue, UX
- **QA Engineer** — Testing y calidad
- **DevSecOps** — Seguridad y compliance
- **DevOps** — CI/CD, Docker, Kubernetes

## Estructura
\`\`\`
agents/
  orchestrator/   # Agente orquestador
  backend/        # Backend Developer
  frontend/       # Frontend Developer
  qa/             # QA Engineer
  devsecops/      # DevSecOps Engineer
  devops/         # DevOps Engineer
shared/
  skills/         # Skills compartidos entre agentes
  prompts/        # Prompts reutilizables
docs/             # Documentación
\`\`\`
